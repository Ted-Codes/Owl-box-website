About me

I am Ted, a highschooler, amateur carpenter, coder, and Eagle scout. This whole project started with me getting into woodworking, then when it came time to do my Eagle Scout Project, I wanted to build an owl box (see picture). So I built my owl box but the plan was always to stream the owl inside, but that is taking a while. So in the meantime I have built this platform to analyze any owl box stream and return data! I also plan to turn the data into a small curriculum for some local schools near me to teach about the enviroment and engineering.
